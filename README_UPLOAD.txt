🚗 FAUZAN RENTCAR 2 - UPLOAD GUIDE

Ikuti langkah berikut untuk mengunggah website ke GitHub Pages:

1️⃣ Login ke akun GitHub kamu
2️⃣ Buka repository: fauzan-rentcar2
3️⃣ Klik tombol "Add file" → "Upload files"
4️⃣ Drag & drop semua isi folder ini (index.html + folder assets)
5️⃣ Scroll ke bawah → klik "Commit changes"
6️⃣ Masuk ke tab: Settings → Pages
7️⃣ Di bagian "Source", pilih:
    - Branch: main (atau gh-pages jika sudah ada)
    - Folder: / (root)
8️⃣ Klik Save

Tunggu sekitar 1 menit, lalu buka:
👉 https://Kamilianurunnadhifah.github.io/fauzan-rentcar2
